# dot-hello-world
How many config files does it take to build a modern Typescript app?
Let's find out!

# Quickstart
1.  Install gulp

    ```shell
    npm install --global gulp
    ```

2. Install dependencies

    ```shell
    npm install
    ```

3. Build it

    ```shell
    gulp
    ```

4. Run it

    ```shell
    node dist/index.js
    ```

    *or*

    ```shell
    npm start
    ```
